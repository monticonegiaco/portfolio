# Portfolio Personal

## Sobre el proyecto

Portfolio personal de [Tu Nombre], Técnico Superior en Computación.

## Stack

- Next.js 15 con App Router
- Tailwind CSS v4
- JavaScript (sin TypeScript)

## Estructura de carpetas

- `app/page.js` — página principal
- `app/components/` — componentes reutilizables (Navbar, Button, Card)
- `app/sections/` — secciones de la página (Hero, Projects, About, Contact)
- `app/globals.css` — estilos globales

## Convenciones

- Componentes con nombres en PascalCase
- Un archivo por componente
- Clases de Tailwind en el mismo archivo del componente
- Sin CSS modules ni styled-components

## Secciones del portfolio

1. Navbar — navegación fija arriba
2. Hero — presentación personal
3. Stack — tecnologías que manejo
4. Projects — grilla de proyectos académicos
5. About — sobre mí
6. Contact — formulario o links de contacto
